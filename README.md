This project analyzes customer shopping behavior using a complete data analytics pipeline.
It includes data cleaning, exploratory data analysis (EDA), SQL-based querying, and an interactive Power BI dashboard to uncover meaningful business insights.

📂 Dataset

The dataset includes customer-related attributes such as:

👤 Age & Gender
💰 Purchase Amount
🛍️ Product Category
🔁 Purchase Frequency
⭐ Review Ratings

Data preprocessing steps were applied to ensure accuracy and consistency.

🛠️ Tools & Technologies
Tool	Purpose
Python (Pandas, NumPy)	Data Cleaning & Processing
Matplotlib & Seaborn	Data Visualization
MySQL	Data Storage & Querying
Power BI	Dashboard Creation
Jupyter Notebook	Development Environment
🔄 Project Workflow
1️⃣ Data Loading
Imported dataset using Pandas
2️⃣ Data Cleaning
Handled missing values
Renamed columns for clarity
Created new features (Age Groups, Purchase Frequency in Days)
3️⃣ Exploratory Data Analysis (EDA)
Analyzed customer demographics
Identified spending patterns
Visualized trends using charts
4️⃣ SQL Analysis (MySQL)
Loaded cleaned data into MySQL
Performed queries using:
GROUP BY
JOIN
AGGREGATE FUNCTIONS
5️⃣ Dashboard (Power BI)
Built interactive visualizations
Displayed KPIs and trends
Dashboard

The Power BI dashboard provides:

Customer segmentation by age and gender
Purchase trends across categories
Frequency of purchases
Key performance indicators (KPIs)

Results & Insights
Identified key customer segments based on age and spending behavior
Observed purchasing patterns across different product categories
Analyzed how frequently customers make purchases
Derived insights to help improve marketing and sales strategies
